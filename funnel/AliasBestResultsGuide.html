AKA / Alias Criminal Records Guide to Best Results

Unlike our standard criminal record search that allows partial name searches, this requires that you enter the exact name. This search uses the social security number to find a name and address history, then uses the name you input and maiden name, married or previously married names, nicknames, and other aliases discovered through the address history to do a criminal record search of our database of over 550 million records.

Input an exact Date of Birth.

Your search is dependent on the input name, birth date, and and social security number and the results may or may not include the input middle initial.

Make sure that you input the first name and last name into the correct fields and double-check to make sure that the date of birth and social security number information you input is correct. Once you you've input the information and go to the next step you'll be taken to a confirmation page which will show the input information. We recommend that you check the information one more time before you proceed. Much like a good woodworker who measure twice to cut once, it's best check twice to search once.
