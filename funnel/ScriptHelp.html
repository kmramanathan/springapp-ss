To enable JavaScript in Internet Explorer 6.0 or higher
  1. Click on Tools at the top, and go down to Internet Options.
  2. Click on the Security tab at the top of the Internet Options window.
  3. Click on the button that says Custom Level.
  4. Scroll down to where it says Active Scripting, and check the button below it that says Enable.
  5. Click the OK button. When it asks you whether you want to change the security settings for this zone, click the YES button.
  6. Click the OK button in the Internet Options window.
  7. Now return to our Landing page.
To enable JavaScript in Netscape Navigator 7.0
  1. Click on Edit at the top, and go down to Preferences (on Macintosh, click on Netscape, then Preferences)
  2. Click on the symbol next to where it says Advanced so that it expands the sub-categories.
  3. Click on Scripts & Plugins.
  4. Find where it says Enable JavaScript for, and check the box below it that says Navigator.
  5. Click the OK button at the bottom.
  6. Now return to our Landing page.
To enable JavaScript in Mozilla 1.3 or higher
  1. Click on Edit at the top, and go down to Preferences.
  2. Click on the symbol next to where it says Advanced so that it expands the sub-categories.
  3. Click on Scripts & Plugins.
  4. Find where it says Enable JavaScript for, and check the box below it that says Navigator.
  5. Click the OK button at the bottom.
  6. Now return to our Landing page.
